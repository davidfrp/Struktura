Struktura

It is fast. Mobile-first. And works on all devices... This is Skruktura. An open-source template for modern websites.

Responsive Design

Build from scratch, and only with the use of HTML and CSS (and a few hours of boredom), have this template been made. With nearly all varibles being either dynamic or written in percents, this template is as made for mobile devices.

Typography

The typography base is Roboto which is made by Google. All fonts in this template uses Roboto.

The headings have a font weight of 300, while the buttons use 600. This makes the headings crispy thin and the buttons bold and daring.
